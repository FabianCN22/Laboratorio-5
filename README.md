# Laboratorio 5

Laboratorio de Programación Bajo Plataformas Abiertas.

Este proyecto implementa estructuras de datos en lenguaje C:

- Arreglos dinámicos
- Listas enlazadas simples
- Listas doblemente enlazadas
- Stack

## Compilar

```bash
make
```

## Ejecutar

```bash
./main
```

## Contenido del laboratorio

El laboratorio se divide en distintas ramas, donde cada una trabaja una estructura de datos diferente.
